# Assets Required

This directory should contain the following assets:

1. **app-icon.png** - Your app icon (512x512px recommended)
   - Should be a square image of your Let's Fit Go app icon
   - Used in the invite page header

2. **app-store-badge.svg** - Apple App Store badge
   - Download from: https://developer.apple.com/app-store/marketing/guidelines/
   - Should be the official "Download on the App Store" badge

3. **google-play-badge.png** - Google Play Store badge
   - Download from: https://play.google.com/intl/en_us/badges/
   - Should be the official "Get it on Google Play" badge

**Note:** Currently, placeholder files exist. Replace them with actual assets before deploying to production.
